# 会員紹介システム 仕様書

## 参考サイト
京都倫理法人会 会員紹介ページ: https://kyoto-rinri.com/members/#voice

## 画面構成

### レイアウト
- **左側**: 絞り込み・検索エリア（サイドバー）
- **右側**: 会員カード一覧（グリッド表示）

### 左側サイドバー機能

#### 1. 並び替え（ラジオボタン）
- ランダム
- 新しい順（登録日降順）
- 古い順（登録日昇順）

#### 2. カテゴリー（チェックボックス）
- 専門サービス（士業,保険,デザイン,議員等）
- 飲食・食品
- 建築・不動産
- 美容・健康
- 製造・ものづくり
- 医療・福祉
- 電気・通信・IT
- 小売・販売
- その他
- 教育・スクール
- 事業承継

#### 3. 所属委員会（チェックボックス）
北上市倫理法人会の場合は、所属単会ではなく「所属委員会」として実装
- 例: 会員拡大委員会、活力朝礼委員会、広報委員会など

#### 4. 検索フィールド
- プレースホルダー: "会社名・氏名を検索"
- テキスト入力による部分一致検索

#### 5. アクションボタン
- **絞り込む**: 選択した条件で会員を絞り込み
- **リセット**: すべての条件をクリアして初期状態に戻す

### 会員カード表示

#### カードレイアウト
- グリッド表示（2〜3カラム、レスポンシブ）
- 各カードに以下の情報を表示:
  - 所属委員会バッジ（ピンク背景）
  - タイトル（会員の声のタイトル）
  - 会社名
  - 氏名
  - 会員写真
  - 詳細ボタン

#### カードデザイン
- 緑の破線ボーダー
- ホバー時にシャドウ効果
- 写真は正方形または縦長の比率

## データベース設計

### members テーブル
| カラム名 | 型 | 説明 |
|---------|-----|------|
| id | int | 主キー |
| name | varchar(100) | 氏名 |
| company_name | varchar(200) | 会社名 |
| title | varchar(200) | 会員の声のタイトル |
| message | text | 会員の声の本文 |
| photo_url | varchar(500) | 会員写真のURL |
| category | varchar(50) | カテゴリー |
| committee | varchar(50) | 所属委員会 |
| sort_order | int | 表示順序 |
| created_at | timestamp | 登録日時 |
| updated_at | timestamp | 更新日時 |

## 管理画面機能

### CRUD操作
1. **一覧表示**: すべての会員を表示、検索・絞り込み可能
2. **新規登録**: 会員情報と写真をアップロード
3. **編集**: 既存会員情報の更新
4. **削除**: 会員情報の削除（確認ダイアログ表示）

### 写真アップロード
- S3ストレージに保存
- 推奨サイズ: 400x400px以上
- 対応形式: JPEG, PNG
