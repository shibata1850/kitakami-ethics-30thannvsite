# ログイン・登録画面実装記録

## 実装内容

### 1. ログイン画面（/login）
- メールアドレスとパスワードの入力フォーム
- 桜色をベースにした洗練されたデザイン
- ログイン成功時にJWTトークンをlocalStorageに保存
- ダッシュボードへのリダイレクト機能
- 新規アカウント作成へのリンク
- トップページへの戻るリンク

### 2. 新規登録画面（/register）
- お名前、会社名、メールアドレス、パスワード（確認含む）の入力フォーム
- パスワードバリデーション（8文字以上、一致確認）
- 登録申請完了画面の表示
- アドミン承認待ちの説明
- ログイン画面へのリンク
- トップページへの戻るリンク

### 3. デザイン特徴
- 桜色（ピンク系）のグラデーションボタン
- カード形式のレイアウト
- ロゴマークの表示
- レスポンシブデザイン対応
- ローディング状態の表示（Loader2アイコン）
- エラーハンドリングとトースト通知

### 4. 技術スタック
- React 19
- Wouter（ルーティング）
- shadcn/ui（UIコンポーネント）
- tRPC（API通信）
- Sonner（トースト通知）
- Lucide React（アイコン）
